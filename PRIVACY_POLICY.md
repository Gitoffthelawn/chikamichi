# Privacy Policy for Chikamichi

Last updated: 2026-05-16

Chikamichi is a browser extension that helps users quickly search and move between browser history, bookmarks, and tabs.

## Data We Access

Chikamichi may access the following browser data to provide its core features:

- Browsing history
- Bookmarks
- Open tabs
- The currently active tab

## How We Use Data

This data is used only to provide extension features, including:

- Fuzzy search for history, bookmarks, and tabs
- Opening and switching tabs
- Running browser search from the extension UI
- Capturing screenshots when the user explicitly chooses that action

## Data Storage

Chikamichi stores user settings and favorite items locally in the browser.

## Data Sharing

Chikamichi does not sell, transfer, or share user data with third parties.

## Remote Code

Chikamichi does not use remote code. All extension code is included in the packaged extension.

## Screenshots and Downloads

If the user chooses a screenshot action, the generated image is saved locally to the user's device using the browser downloads feature.

## Contact

For questions about this privacy policy, please contact the project maintainer through the GitHub repository:
